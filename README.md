###react-native-movies

React Native Expo app to learn essential skills for mobile development along the way.
Tutorial from: [Youtube Javascript Mastery](https://www.youtube.com/watch?v=f8Z9JyB2EIE)

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
